# AnshiChatRobot ‚Äì A Sanskari Baby Girl ChatBot for Telegram

![AnshiChatRobot Banner](https://files.catbox.moe/8ic7jp.jpg)

> **"Main Anshi hoon, ek sanskari aur sweet baby girl chatbot!"**

AnshiChatRobot is an AI-powered chatbot who talks like a real 5-year-old sanskari baby girl. She brings sweetness and tradition to your groups, answers questions using AI, scans photos for answers, and even blocks bad language.

---

## ü™Ñ Features

- **Real Human-like Chatting** (OpenAI powered)
- **Auto Chat in Group** ‚Äì Ignores replies and replies only to random messages
- **Abusive Filter** ‚Äì Blocks bad words
- **Sanskari Baby Girl Personality**
- **/ask** ‚Äì Ask anything to AI
- **/scan** ‚Äì Send a photo with a question and get an answer
- **/chatbot on/off** ‚Äì Control the AI replies
- **/clone** ‚Äì Clone this bot with your own token
- **Modular Files** ‚Äì Clean and expandable structure
- **MongoDB Storage** ‚Äì For chats and users
- **Stylish Start Panel** ‚Äì With inline buttons and image
- **Logger** ‚Äì Logs user information when starting the bot
